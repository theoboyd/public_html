Full Screen Instructions

Windows

Mac OS
   Install SIMBL
   Add megazoomer.bundle to ~/Library/Application Support/SIMBL/Plugins
   Open a SIMBL compatible web browser and navigate to this page*
   Click the full screen link above
   Press ⌘+⏎


*You may need to restart any open applications like your web broswer and Finder (killall Finder) after first install of megazoomer.bundle
