What is that coloured box?

It's a High Capacity Colour Barcode (HCCB) representing this website's URL - and it's readable by many smartphones and also certain specialised scanners and software.

To use the one on this site, click the Using HCCBs tab above. Or read more below.

What does the website colour scheme mean?

The website uses three colours for links and tags. View the table.

Item Meaning
Link style tag Not used
Info style tag Timestamp or highlighted text
Error style tag Details about an error that has ocurred
Gold style tag A promoted or important update
Link style hyperlink A (usually internal) hyperlink
Info style hyperlink A link for more information in the current context
Error style hyperlink A link to correct an error in the current context
Gold style hyperlink Not used
